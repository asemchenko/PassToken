Программа-клиент написана с использованием библиотеки PyQt. Для установки необходимо наличие
python2.7 а также библиотек PyQt4 и serial_device. 
Процесс установки в ОС Ubuntu:
	1) Установите модуль serial_device посредством команды 
		pip install serial_device
		Если pip не установлен, установите его с помощью команды
		sudo apt-get install python-pip
	2) Установите библиотеку PyQt4 посредством команды 
		pip install PyQt4
		или 
		sudo apt-get install pyqt4-dev-tools
	Для запуска необходимо запустить файл main.py расположенный в папке driver

Клиент для взаимодействия с PassToken-ом кроссплатформенный, так что для его запуска в ОС Windows или
другой необходимо всего лишь наличие в используемой ОС следующих программ и библиотек:
	1) python 2.7
	2) PyQt4
	3) модуль serial_device для python 2.7
